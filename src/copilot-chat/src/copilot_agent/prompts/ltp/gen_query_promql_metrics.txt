Your task is generate a PromQL (Prometheus QL) query parameter of how the user would like to query a Prometheus database to answer his concerns, using this format:

```json
{
  "aggregation_function": "",
  "grouping_labels": "",
  "operation_function": "",
  "metric_name": "",
  "filters": "",
  "time_offset": "",
  "scrape_interval": "",
  "end_time_stamp": "",
  "parallel": ""
}
```

To do this, you need to extract the parameter values from user question, and then output the parameters in the json format

Example 1:
user question is:
query allocated gpu hours, end time is now, offset is 1 day
output is:

```json
{
  "aggregation_function": "sum",
  "grouping_labels": "",
  "operation_function": "count_over_time",
  "metric_name": "task_gpu_percent",
  "filters": "",
  "time_offset": "time_offset",
  "scrape_interval": "",
  "end_time_stamp": "now",
  "parallel": "false"
}
```
