We are pleased to announce the official release of **Lucia Training Platform v1.0.0**!

## Lucia Training Platform v1.0.0 Release Notes

This inaugural release establishes Lucia Training Platform as a comprehensive AI training solution, built on the foundation of OpenPAI with significant enhancements and customizations for enterprise AI workloads.

## Platform Features & Stability
- **VMSS Deployment Scripts for MI300**: Updated Virtual Machine Scale Set deployment scripts with MI300 GPU support and kubelet bug fixes
- **AMD Device Plugin and AMDGPU Module**: Fixed launch order issues between AMD device plugin and AMDGPU module loading
- **RAID Data Loss Prevention**: Resolved critical RAID data loss issues that occurred after system reboots
- **Local Disk Mount Optimization**: Fixed local disk mounting into containers for significantly faster data loading
- **Production Job Priority Restrictions**: Implemented priority restrictions for production jobs to ensure resource allocation
- **User Logs Backup System**: Automated daily backup of user logs to blob storage with cordon trigger functionality
- **Large-Scale SSH Crash Resolution**: Updated OpenPAI-runtime image to resolve SSH crashes in large-scale deployments
- **Storage Cache Management**: Added refresh API to clean storage cache when new Persistent Volumes (PV) or Persistent Volume Claims (PVC) are added
- **Production Job Notifications**: Implemented automated email notifications for production jobs to specific user groups

## Job Reliability & Monitoring
- **AMD GPU Hang Detection**: Implemented automatic detection and cordon rules for AMD GPU hang issues during runtime
- **Auto Node Cordon/Uncordon**: Automated node cordoning and uncordoning based on single node validation job logs
- **Cordoned Node Job Support**: Enabled job execution on selected cordoned nodes for improved resource utilization
- **Resource Availability Metrics**: Comprehensive monitoring for count of available, used, and User Acceptance (UA) resources
- **AMD GPU Monitoring**: Detailed metric monitoring specifically for AMD GPU performance and status tracking

## User Experience
- **Job Priority Guidance**: Comprehensive documentation and guidance for job priority management
- **Virtual Cluster (VC) Allocation Process**: Detailed process documentation for VC allocation and management
- **Issue Handling Process**: Streamlined documentation for issue reporting and resolution workflows
- **Homepage Revision**: Complete revision of the homepage with acknowledgment of OpenPAI's great contribution
- **Platform Branding**: Updated all titles and references from OpenPAI to Lucia Training Platform (LTP) throughout the web portal
