We are pleased to announce the official release of **Lucia Training Platform v1.2.0**!

## Lucia Training Platform v1.2.0 Release Notes

This release introduces significant new features, enhanced reliability monitoring, improved user experience, and strengthened security measures.

## Platform Features & Stability
- **VC Admin Job Stop Support**: Virtual Cluster administrators can now stop jobs directly from the portal
- **Gateway for Inference Jobs with External IP**: Enhanced inference job connectivity with external IP gateway support
- **Alive Cluster Only Portal Support**: Portal now displays only active clusters for improved user experience
- **Docker-in-Job Support**: Enhanced job execution capabilities with Docker support within jobs
- **History Clean Support**: Automated cleanup functionality for job history management
- **Local Storage Design**: Comprehensive design for local storage service implementation
- **Local Storage Service Implementation**: Core implementation of local storage functionality
- **Node Recycling Pipeline Integration**: Seamless integration with node recycling processes
- **Job-exporter Crash Fix**: Resolved CUDA version mismatch issues causing job-exporter crashes
- **Rest-server Config Refresh**: Fixed configuration refresh issues when updating user settings
- **Blob Mount and Azure Copy Token**: Resolved blob mount failures and Azure copy token issues

## Job Reliability & Monitoring
- **Detector Design**: Implemented failure detection design in alert-manager
- **Detector Implementation**: Complete implementation of automatic failure detection system
- **Job Summary and React Time Recording**: Enhanced job monitoring with summary and reaction time tracking
- **Auto Pipeline for Prometheus Deployment**: Automated Prometheus deployment pipeline
- **Certificate Expiration Alert**: Proactive alerting system for certificate expiration management

## User Experience
- **Webportal Plugin for Copilot**: Added webportal plugin integration for Copilot functionality
- **Copilot Backend Support**: Comprehensive backend support for Copilot features
- **Availability, Reliability, Productivity Metrics**: Enhanced dashboard with comprehensive platform metrics
- **MTBI Table for VM and Node**: Added Mean Time Between Incidents (MTBI) tracking for virtual machines and nodes

## Security
- **Vulnerable Container Image Updates**: Ongoing updates to address security vulnerabilities in container images
- **AKS Upgrade**: Kubernetes service upgrade for enhanced security and performance
