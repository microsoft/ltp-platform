We are pleased to announce the official release of **Lucia Training Platform v1.3.0**!

## Lucia Training Platform v1.3.0 Release Notes

This release brings significant improvements across platform stability, inference capabilities, user experience, and security enhancements.

## Platform Features & Stability
- **PostgreSQL Database Migration**: Migrated PostgreSQL database to Azure disk for improved performance
- **Blobfuse Version Upgrade**: Upgraded blobfuse version to support non-empty cache folder
- **SSH-Proxy and Utilization Reporter**: Integrated ssh-proxy and utilization reporter services
- **Local Disk Support for Prometheus**: Added local disk support for Prometheus deployment
- **H200 GPU Support**: Added H200 GPU support for VMSS provisioning scripts
- **Kubernetes Deployment Scripts**: Updated Kubernetes deployment scripts for Kubespray compatibility on Bare Metal Machines

## Inference Plugin
- **Output Streaming Support**: Added support for output streaming in web portal
- **Long-Context Thinking Output Folding**: Implemented long-context thinking output folding support
- **Inference Model Proxy Service**: Deployed inference model proxy service for improved model serving
- **Enhanced Chat Plugin**: Messages now display as markdown and copy as plain text

## Tools
- **Administration Tools**: Added comprehensive administration tools for LTP service management

## Security
- **Dockerfile Updates**: Upgraded Dockerfiles with latest system updates
- **Docker-in-Docker Support**: Added Docker-in-Docker (dind) support for Webportal

## User Experience
- **Code Refactoring**: Code refactoring for improved maintainability
- **Copilot SGLANG/OpenAI Interface**: Integrated Copilot SGLANG/OpenAI Interface backend support
- **Copilot Dashboard Metrics**: Added Copilot support for Dashboard Metrics visualization
- **User Feedback Loop**: Implemented User Feedback Loop design and functionality
- **VC User Group Authentication**: Added VC based user group membership authentication

## License
- **Microsoft License Headers**: Added Microsoft license headers to examples directory
- **Microsoft License Headers**: Added Microsoft license headers to contrib directory
- **Microsoft License Headers**: Added Microsoft license headers to deployment directory
- **Microsoft License Headers**: Added Microsoft license headers to src directory
