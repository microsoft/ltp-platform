We are pleased to announce the official release of **Lucia Training Platform v1.1.0**!

## Lucia Training Platform v1.1.0 Release Notes

This release introduces new inference capabilities, enhanced stability improvements, comprehensive monitoring systems, and significant security enhancements.

## Platform Features & Stability
- **Docker Build Pipeline for Megatron-LM with MI300**: Implemented automated Docker build pipeline supporting MI300 GPU architecture
- **Inference Job Submission Support**: Added comprehensive support for inference job submission and management
- **Blob Mount Crash Issue Resolution**: Fixed critical blob mount crash issues affecting platform stability
- **Continuous Job Retry and Scheduling**: Enhanced job retry mechanisms and scheduling during disk full scenarios
- **SSH Access Management**: Disabled SSH access for all users to enhance security posture
- **Open-Source Process**: Established comprehensive open-source contribution process for LTP

## Job Reliability & Monitoring
- **Runtime Hardware Issue Category Summary**: Implemented comprehensive categorization and summary of runtime hardware issues
- **Automatic Node Processing Pipeline Design**: Complete design and implementation of automated node processing workflows
- **Automatic UA Ticket Submission**: Automated pipeline for submitting User Acceptance tickets for node issues
- **Automatic VM Restart and Validation**: Automated VM restart and validation processes for improved reliability
- **Node Recycle Recording Support**: Enhanced data pipeline with comprehensive node recycling recording capabilities

## User Experience
- **User Question Collection**: Implemented system to collect and analyze frequent user questions and solution flows
- **Multi-Backend Model Support**: Enhanced CoPilot to support different backend models (LTP and customer models)
- **Kusto Database Integration**: Comprehensive Kusto database implementation for action status tracking, node status monitoring, and job status analytics
- **Availability, Reliability, Productivity Metrics**: Enhanced dashboard with comprehensive platform performance metrics

## Security
- **OS, Linux and Python Package Upgrades**: Forced upgrades of operating system, Linux, and Python packages to address security vulnerabilities
- **Golang/Node.js Package Upgrades**: Updated Golang and Node.js packages to latest secure versions
- **Registry Security**: Disabled and replaced unapproved registries (non-ACR/MCR) on LTP platform
